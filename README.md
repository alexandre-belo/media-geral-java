Sistema de Avaliação em Java

Projeto desenvolvido para praticar conceitos de programação orientada a objetos utilizando Java.

Descrição:

O sistema permite cadastrar notas de um aluno, calcular automaticamente a média e verificar se ele foi aprovado com base em critérios definidos pelo usuário.

A aprovação considera:

- Média mínima (nota de corte)
- Frequência mínima

O programa também realiza validação de dados, garantindo que:

- As notas estejam entre 0 e 10
- A frequência esteja entre 0 e 100

Funcionalidades:

- Cadastro dinâmico de notas
- Cálculo automático da média
- Verificação de aprovação
- Validação de entrada de dados

Estrutura do Projeto:

Aluno.java  
Responsável por armazenar as notas, frequência e regras de aprovação.

Main.java 
Responsável pela interação com o usuário e execução do programa.

Conceitos Praticados

- Classes e objetos
- Construtores e sobrecarga
- Encapsulamento
- Listas dinâmicas com ArrayList
- Estruturas de repetição
- Condições lógicas
- Entrada de dados com Scanner

Como Executar

1. Compile os arquivos Java.
2. Execute a classe `Main`.
3. Insira as notas, frequência e critérios de aprovação conforme solicitado no terminal.

Objetivo

Este projeto foi desenvolvido com foco em aprendizado e prática de fundamentos da linguagem Java e programação orientada a objetos.
